## The 3-Step Process

```
Step 1: GitHub          Step 2: Render          Step 3: Done!
Upload your code   →    Deploy online      →    App is live!
   (5 minutes)            (10 minutes)          (Automatic)
```

## Step 1: Upload Code to GitHub

**Read this file:** `GITHUB_SIMPLE_GUIDE.md`

**Quick version:**
1. Download GitHub Desktop
2. Create GitHub account
3. Add your project folder
4. Click "Publish"

**That's it!** No confusing commands.

---

## Step 2: Deploy to Render

**Read this file:** `DEPLOY_TO_RENDER.md`

**Quick version:**
1. Create Render account (use GitHub to sign up)
2. Connect your repository
3. Fill in settings (copy from guide)
4. Click "Create Web Service"
5. Wait 10 minutes

**Done!** Your app is online.

## What Happens After Deployment?

### ✅ Your app will:
- Be accessible from anywhere with internet
- Have HTTPS (secure connection)
- Auto-restart if it crashes
- Auto-deploy when you push changes to GitHub

### ⚠️ Free tier limitations:
- Sleeps after 15 minutes of no activity
- Takes 30 seconds to wake up on first request
- 750 hours/month (enough for testing)

Upgrade to $7/month in Render dashboard

- "Authentication failed" → Sign out and sign in again
