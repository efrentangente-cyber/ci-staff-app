# Quick Start Guide - CI Checkbox Summary

## What's New?

A new checkbox summary page that shows ALL checkboxes from the 5-page wizard in one organized page.

## How to Use (CI Staff)

### Step 1: Review Application
1. Login as CI staff
2. Go to CI Dashboard
3. Click on an assigned application
4. You'll see the **Review Page** with:
   - Application details
   - All uploaded images/documents
   - "Fill Interview Checklist" button

### Step 2: Fill Checkbox Summary
1. Click **"Fill Interview Checklist"** button
2. You'll see the **Checkbox Summary Page** with:
   - 14 organized sections
   - 115+ checkboxes
   - Progress indicator on the right
3. Fill all checkboxes by clicking them
4. Watch the progress bar update in real-time
5. When done, click **"Proceed to 5-Page Form"**

### Step 3: Complete 5-Page Wizard
1. The wizard opens with:
   - ✅ All checkboxes already filled (from summary)
   - ✅ Text fields auto-filled (from OCR/AI)
2. Review the auto-filled data
3. Complete any remaining fields
4. Add your signature
5. Upload photos (optional)
6. Submit

7. **Credit Checking** - Electric, Water, Banks, Cooperatives

9. **Capacity** - DEIR, NTHP within limits
10. **Character** - Payment history, feedback from institutions

13. **Business** - Permits, legal nature
14. **Recommendations** - Approve/Disapprove/Defer for each position

## Tips

- **Progress Bar**: Shows your completion percentage
- **Green Highlight**: Auto-filled items are highlighted in green
- **Save Time**: Fill all checkboxes in one page instead of navigating 5 pages
- **No Data Loss**: Your checkbox selections are saved automatically
- **Mobile Friendly**: Works on tablets and phones

## Troubleshooting

### Checkboxes not appearing in wizard?
- Clear browser cache
- Make sure JavaScript is enabled
- Try a different browser

### Progress bar not updating?
- Refresh the page
- Check browser console for errors

### OCR data not loading?
- Make sure LPS uploaded images
- Check that OCR credentials are set in environment variables
- OCR and checkbox data work independently
